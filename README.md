This repository hosts a simple static website that tracks and displays the percentage of flights flown in First Class while traveling non-revenue.

The site is built with plain HTML, CSS, and optional JavaScript, and is deployed using GitHub Pages.

## Purpose
This project is intended for personal use as a lightweight dashboard and learning exercise for static site hosting and front-end layout.

## Tech Stack
- HTML5
- CSS3
- JavaScript (optional)
- GitHub Pages

## Deployment
The site is deployed via GitHub Pages from the `main` branch.

## Notes
- No backend or server-side logic is used.
- All calculations are performed client-side.
